## Fonts

This project expects the Figma display font to be available locally:

- `PasticheGrotesqueTrial-Regular.woff` (weight 400)
- `PasticheGrotesqueTrial-Medium.woff` (weight 500)

Put those files in this folder so the site can match Figma typography as closely as possible.

If they are missing, the site will fall back to system serif fonts.
